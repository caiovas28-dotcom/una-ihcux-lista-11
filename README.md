Proposta de Valor

O ReCiclo é um aplicativo que conecta pessoas que desejam doar objetos com outras que precisam deles. O objetivo é facilitar a doação, incentivar o reaproveitamento e reduzir o desperdício.

Descrição das Telas

Tela 1 — Home
Exibe os itens disponíveis. Permite navegar entre categorias, acessar filtros e outras telas do aplicativo.

Tela 2 — Filtros
Permite filtrar itens por categoria e distância.

Tela 3 — Detalhes do Item
Mostra imagem, descrição, localização e informações do item. Possui botão para iniciar conversa.

Tela 4 — Cadastro de Item
Permite adicionar um novo item com imagem, descrição, qualidade e localização.

Tela 5 — Chat
Tela de conversa entre usuários para combinar a retirada do item.

Tela 6 — Perfil
Mostra dados do usuário e conquistas, como o Selo Eco.

Jornada do Usuário

O usuário acessa a tela inicial, visualiza itens ou aplica filtros, seleciona um item, vê os detalhes e inicia uma conversa. Após combinar a retirada, a ação é concluída e refletida no perfil.


Observações

O protótipo foi desenvolvido no Miro e segue o modelo de baixa fidelidade, focando na estrutura e navegação do sistema.
